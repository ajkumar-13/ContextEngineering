# References

Master bibliography for the series. Citations are grouped by post; full URLs appear in each entry. Where a source is referenced from multiple posts, it is listed once and cross-linked.

> Citation style: author(s), title, venue or publisher, year. Online sources include the URL and the access date used at writing time. Where relevant, a one-line note explains which point the source supports.
