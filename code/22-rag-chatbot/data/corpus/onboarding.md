# Onboarding

A new Acme workspace is provisioned in under five minutes. The administrator
invites teammates by email; each invitee receives a setup link valid for 72
hours. After that, the link expires and a fresh invite is required.

Single sign-on (Google, Microsoft, Okta) is available on the Business and
Enterprise plans. Free and Pro plans use email-and-password authentication
with optional TOTP-based two-factor.

The default workspace timezone is taken from the administrator's browser at
sign-up; it can be changed under Settings → Workspace → Locale at any time.
